# Rust-echo
The echo UNIX utility written in Rust

# Usage
```shell
$ echo <TEXT> [-n]
```

 - `<TEXT>` Stands for any input text 

Options:
  - `-n`             Omits the final newline  
  - `-h`, `--help`     Print help  
  -  `-V`, `--version`  Print version  

# Build
```shell
$ cargo build
```
